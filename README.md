# forensic-soil-chemometrics
Python scripts for the forensic discrimination and provenancing of soil samples using FTIR spectroscopy and multivariate statistical analysis (PCA/LDA and PLS)

Forensic Soil Analysis & Chemometric Modeling
Project Overview
This repository contains the analytical workflow and Python scripts developed for the forensic examination of soil samples. The project focuses on using Fourier Transform Infrared (FTIR) spectroscopy to characterize soil composition and applying Chemometrics to differentiate samples based on their geographical or environmental origin.

Key Features
Data Preprocessing: Baseline correction (AsLS), normalization, and smoothing of spectral data.
Dimensionality Reduction: Implementation of Principal Component Analysis (PCA) to identify variance in soil minerals and organic matter.
Classification: Linear Discriminant Analysis (LDA) for forensic sample grouping and source attribution.
Visualization: Generation of 2Dscore plots and loading plots to interpret spectral markers.
Implementation of partial Least Squares using cross validation.

Applications
This toolset is designed for forensic geoscience applications, including:
Crime scene linking (comparing soil from footwear/vehicles to a known location).
Environmental monitoring and soil provenancing.
Enhancing judicial outcomes through objective, data driven evidence.
